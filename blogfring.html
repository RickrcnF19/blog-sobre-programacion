1. ¿Qué es programar?

Programar es dar instrucciones a una computadora para que realice tareas. Escribir programas implica usar un lenguaje (JavaScript, Python, Java, etc.), datos y lógica.

2. Conceptos básicos

  • Variable: espacio para guardar información.
  • Función: bloque de código reutilizable que realiza una acción.
  • Condicional: permite tomar decisiones (if / else).
  • Bucle: repite acciones (for, while).

3. Ejemplo: función en JavaScript

Esta es una función que toma un nombre y devuelve un saludo:
